# 📋 要件定義書

> **Project**: BBQ (Bird Battle Quest)  
> **Version**: 0.1.0  
> **Last Updated**: 2025-12-13

---

## 1. プロジェクト概要

### 1.1 目的
FF6のクラシックな2D RPG体験とオクトパストラベラーの戦略的バトルシステムを融合させた、
個人開発規模で実現可能なモダンなピクセルアート RPG。

### 1.2 ビジョン
- FF6風のタイルマップ移動で懐かしさを演出
- オクトパストラベラー風のブレイク/シールドシステムで戦略性を実現
- 育成要素（レベルアップ、武器合成）で成長の喜びを提供

---

## 2. ユーザー要件

### 2.1 ターゲットユーザー
- **プライマリ**: クラシックJRPGファン、ピクセルアートゲーム愛好家
- **セカンダリ**: カジュアルにRPGを楽しみたいプレイヤー

### 2.2 利用環境
| デバイス | 推奨 | 対応 |
|----------|------|------|
| スマートフォン | ✅ | ✅ (PWA) |
| タブレット | ✅ | ✅ |
| PC | ✅ | ✅ |

### 2.3 ユーザーストーリー

#### US-001: マップ移動
> プレイヤーとして、フィールドを自由に歩き回りたい。なぜならRPGの探索を楽しみたいから。

**受け入れ条件:**
- [x] WASD/矢印キーで4方向移動できる
- [x] カメラがプレイヤーに追従する
- [ ] 障害物と衝突判定がある

#### US-002: ランダムエンカウント
> プレイヤーとして、歩いていると敵と遭遇したい。なぜならバトルを楽しみたいから。

**受け入れ条件:**
- [x] 一定歩数で敵とエンカウントする
- [x] エンカウント時にフラッシュ演出がある
- [x] バトル画面に遷移する

#### US-003: ターン制バトル
> プレイヤーとして、コマンドを選んで敵と戦いたい。なぜなら戦術を考える楽しさがほしいから。

**受け入れ条件:**
- [x] たたかう/ぼうぎょ/にげるコマンドがある
- [x] HPダメージ計算がある
- [x] 勝利/敗北判定がある

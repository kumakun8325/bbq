## 📋 タスクの追加方法

新しいタスクを追加する場合:

1. 該当セクションにタスクを追加
2. 優先度に応じて並び替え
3. 着手時に「進行中」セクションに移動
4. 完了したら「完了タスク」セクションに移動

### タスクフォーマット
```markdown
- [ ] **TASK-XXX**: タスク名 - 簡潔な説明
```
